# Jardim Cigano da Meg 🔮

Projeto web desenvolvido para agendamento de consultas espirituais online, com área administrativa para gerenciamento dos atendimentos.

## 🌐 Sobre o projeto

O Jardim Cigano da Meg é uma plataforma de agendamento onde o cliente pode visualizar os serviços disponíveis, escolher uma data e horário, preencher seus dados e receber as informações de pagamento via Pix.

O projeto também possui uma área administrativa para acompanhar os agendamentos realizados.

## 🚀 Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS
- Supabase
- React Router
- Vercel

## ✨ Funcionalidades

- Página inicial personalizada
- Listagem de serviços
- Agendamento online
- Calendário com dias úteis
- Horários disponíveis de segunda a sexta
- Integração com Supabase
- Tela de confirmação com dados de pagamento
- Área administrativa
- Login administrativo
- Listagem de agendamentos
- Atualização de status dos atendimentos
- Layout responsivo

## 📌 Regras de atendimento

- Atendimento de segunda a sexta
- Horários disponíveis:
  - 10h às 12h
  - 13h às 19h
- Atendimento realizado por mensagem após confirmação do pagamento
- Envio de fotos das tiragens e áudios explicativos
